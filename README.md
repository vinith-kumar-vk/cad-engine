# CAD Engine

A forward-thinking EdTech and BIM consulting company website redefining digital learning in the Architecture, Engineering, and Construction (AEC) industry.

## Features
- CAD & BIM Training courses (AutoCAD, Revit, Navisworks)
- CAD Design & BIM Project Services
- Manpower Deployment
- Modern, responsive UI with interactive animations

## Tech Stack
- HTML5, Bootstrap, CSS3, JavaScript
- PHP (Server-side rendering)
